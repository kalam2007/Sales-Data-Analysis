# 📊 Sales Data Analysis

## 📌 Overview

This project analyzes sales data to identify top-performing products, regions, and revenue trends. It helps businesses understand customer buying behavior and optimize sales strategies.

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Excel (for dashboard)
* Power BI

## 📂 Dataset

Synthetic dataset created within the script for demonstration purposes.

## 📊 Key Analysis Performed

* Product-wise sales analysis
* Region-wise performance analysis
* Monthly sales trend analysis

## 📈 Key Insights

* Identified top-performing products contributing maximum revenue
* Analyzed region-wise sales performance
* Detected peak sales months for better planning

## 📊 Visualizations

* Monthly Sales Trend Line Chart

## ▶️ How to Run

```bash
pip install pandas matplotlib
python sales_analysis.py
```
